# SPECTRE

**High-Performance Commerce Interface**

Designed & Engineered by **Alfaz Mahmud Rizve**.

## Overview
SPECTRE is a next-generation e-commerce frontend designed to demonstrate "Agentic AI" capabilities in modern web development. It features a cinematic, immersive user experience ("Cyberpunk/Stealth" aesthetic) built with performance and motion at its core.

## Tech Stack
- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS + Custom Design System
- **Animation:** Framer Motion + GSAP Concepts
- **3D/Canvas:** HTML5 Canvas for Sequence Rendering
- **State Management:** Zustand (Persisted Cart & Settings)
- **Authentication:** NextAuth.js
- **Database:** JSON-based mock DB (File System)

## Key Features
- **Cinematic Product Sequence:** Scroll-controlled frame interpolation.
- **Agentic Admin Dashboard:** Command-center style inventory management.
- **Secure Terminal Checkout:** immersive, form-based checkout flow.
- **Dynamic Routing:** Individual product detail pages.

## Deployment
Deployed on Vercel.
